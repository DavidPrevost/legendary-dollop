# Python - Level Structure

## Overview

- **Total Levels**: 5 (0-4)
- **Total Concepts**: 465
- **Estimated Total Time**: 50-70 hours

---

## Level 0: Curious (First Scripts)

**Concepts**: 1-95 (95 concepts)
**Modules**: 1-3
**Time Estimate**: 8-12 hours

### Description
Your first steps in Python programming. Learn to write simple scripts, work with text and numbers, and use lists. By the end, you'll be writing useful small programs.

### What You'll Learn
- Running Python code
- Variables and basic types
- Strings and formatting
- Lists and list comprehensions
- Basic operations and iteration

### Prerequisites
- None (entry level)
- Command Line Mastery recommended

### Comparison Band
Novice - No prior Python experience required.

### Exit Criteria
- Write and run Python scripts
- Use variables and basic types correctly
- Manipulate strings with methods and f-strings
- Create, modify, and iterate over lists
- Use list comprehensions for transformation

---

## Level 1: Explorer (Control and Functions)

**Concepts**: 96-210 (115 concepts)
**Modules**: 4-6
**Time Estimate**: 12-16 hours

### Description
Master data structures and learn to organize code with functions. Understand conditionals, loops, and how to write reusable code with functions and decorators.

### What You'll Learn
- Dictionaries and sets
- Control flow (if/for/while)
- Exception basics
- Functions and scope
- Decorators

### Prerequisites
- Level 0 completion

### Comparison Band
Novice → Competent - Can write structured programs with functions.

### Exit Criteria
- Choose appropriate data structure for task
- Write conditional and loop logic
- Handle basic exceptions
- Create functions with various argument types
- Understand and use simple decorators

---

## Level 2: Tinkerer (Classes and Files)

**Concepts**: 211-325 (115 concepts)
**Modules**: 7-9
**Time Estimate**: 12-16 hours

### Description
Learn object-oriented programming and file operations. Create classes, use inheritance, and read/write various file formats. Understand modules and packages.

### What You'll Learn
- Classes and objects
- Inheritance and polymorphism
- Magic methods
- Modules and packages
- File I/O for text, CSV, JSON

### Prerequisites
- Level 1 completion

### Comparison Band
Competent - Can design classes and work with files.

### Exit Criteria
- Create classes with appropriate methods
- Use inheritance and composition
- Import and use standard library modules
- Read/write text, CSV, and JSON files
- Organize code into modules

---

## Level 3: Builder (Data and Automation)

**Concepts**: 326-420 (95 concepts)
**Modules**: 10-12
**Time Estimate**: 10-14 hours

### Description
Handle errors gracefully, work with APIs and databases, and automate tasks. Build practical scripts that interact with external systems.

### What You'll Learn
- Comprehensive error handling
- HTTP requests and APIs
- SQLite databases
- Automation scripts
- Command-line tools

### Prerequisites
- Level 2 completion

### Comparison Band
Competent → Proficient - Can build automation tools and data pipelines.

### Exit Criteria
- Handle exceptions appropriately
- Consume REST APIs
- Query SQLite databases
- Create automation scripts with logging
- Build CLI tools with argparse

---

## Level 4: Maker (Testing and Advanced)

**Concepts**: 421-465 (45 concepts)
**Modules**: 13-15
**Time Estimate**: 8-12 hours

### Description
Write tested, maintainable code. Learn debugging techniques, type hints, and basic async programming. Polish projects for sharing.

### What You'll Learn
- Unit testing with pytest
- Debugging and profiling
- Type hints and validation
- Async programming basics
- Project organization

### Prerequisites
- Level 3 completion

### Comparison Band
Proficient → Expert - Can write tested, production-quality code.

### Exit Criteria
- Write comprehensive tests with pytest
- Debug code using pdb and IDE tools
- Add type hints to functions
- Understand when to use async
- Structure projects for maintainability

## Recommended Path

For makers wanting to automate their workflow:

1. **Level 0-1**: Core language skills
2. **Level 2**: File handling for data processing
3. **Level 3**: Automation and APIs (main goal)
4. **Level 4**: Optional for production-quality code

Total focused path: ~35-45 hours to practical automation skills.
