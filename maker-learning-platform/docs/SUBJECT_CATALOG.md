## Subject Relationships

### Foundation Dependencies
Most subjects benefit from these foundations:
- **Project Foundations** → All subjects (documentation patterns)
- **Command Line Mastery** → Most technical subjects
- **Git/GitHub** → All coding projects

### Integration Opportunities
Key combinations that unlock powerful projects:
- Python + SQL + Databases → Data automation
- HTML/CSS + JavaScript + Python → Web dashboards
- Docker + Linux Administration + Networking → Homelab
- Dart/Flutter + Databases + API Basics → Mobile apps
- Bash + Git + Testing → DevOps pipelines
